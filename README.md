<h1 align="center">Concurrency</h1>

<p align="justify">
    Welcome to the <b>Concurrency</b> repository! This repository provides hands-on demos covering different ways to
    execute tasks concurrently using <b>Runnable</b> and <b>Callable</b> with various thread pool executors. It also
    explores synchronization techniques using <b>Lock</b> and <b>Atomic</b> classes, ensuring efficient and thread-safe
    execution in multi-threaded applications.
</p>

<hr>

<h3 align="left">üìå Table of Contents</h3>
<ol align="left">
    <li>Runnable without Concurrency</li>
    <li>Runnable using SingleThreadExecutor</li>
    <li>Runnable using FixedThreadPool</li>
    <li>Runnable using CachedThreadPool</li>
    <li>Runnable using CachedThreadPool with Future</li>
    <li>Callable using CachedThreadPool with Future</li>
    <li>Synchronization using Lock and Atomic class</li>
    <li>Runnable using ScheduledThreadPool</li>
</ol>
